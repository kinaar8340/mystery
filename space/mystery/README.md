**Browser demo** of the Mystery research notebook: near-Pythagorean triangle φ²+e²≈π², holonomy-gap scaling B(κ), and 30-60-90 / 3-6-9 comparisons.

No install required — use the **App** tab. The UI keeps the **CLI terminal + programmable keypad** from [orbital-braille-vqc](https://huggingface.co/spaces/kinaar111/orbital-braille-vqc) with a gold/purple mystery theme.

> **Numerical probe** — matplotlib figures generated in-browser. Full 11-probe suite: [`run_all.py`](https://github.com/kinaar8340/mystery) locally.

---

## In 60 seconds

1. Open the **App** tab.
2. Use the **matrix terminal** and **d-pad** (or prog keys 02–05) to navigate status, demo scope, build info, and help.
3. Tune **κ** on the slider (κ_doc = 0.85 marked).
4. Click **Run analysis** → read metrics, view triangle and κ-sweep plots.
5. Open **Figures** for reference plots from the probe suite.
